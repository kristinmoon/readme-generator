
  # LookUp
  [![License: WTFPL](https://img.shields.io/badge/License-WTFPL-brightgreen.svg)](http://www.wtfpl.net/about/)

  ## Description
  LookUp is an astronomy-based app that allows the user to use their browser location at a chosen date and time to see a list of planetary objects that will be above them. Each search result includes a corresponding photo of the body from NASA Images.

  ## Installation
  This project does not require any installation on the user’s end. You will need to enable the browser’s geolocation to utilize our app. If you would rather, you can also enter your latitude and longitude into the application manually.

  ## Usage
  Enjoy finding out what bodies will be above you at any given time and location.
